# Railway Web App

A simple full-stack railway website with:
- Backend built with Express
- Frontend served as static HTML/CSS/JS
- User registration stored in `data/users.csv`
- Real-time updates using Socket.IO

## Setup

1. Open the project folder in your terminal:
   ```bash
   cd c:\Users\LENOVO T460\Desktop\begin
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the app:
   ```bash
   npm start
   ```
4. Open `http://localhost:3000` in your browser.

## Features

- Register passengers with name, email, and phone
- Persist user data in a CSV file
- Display users in real time when new registrations arrive
